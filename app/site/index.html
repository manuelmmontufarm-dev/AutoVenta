Cómo se conecta todo

Una sola base de código, tres superficies. El backend siempre trae TODO; las fases deciden qué se muestra en cada cliente.

1 · Tu laboratorio

Staging

Donde trabajas y pruebas. WhatsApp de prueba (tester). Todas las fases encendidas.

autoventa-staging.up.railway.app
2 · Lo que entregas

Cliente · Depot Tire

Su WhatsApp Business, su base de datos limpia, su clave. Ve solo las fases que le enciendes.

autoventa-depottire.up.railway.app
3 · Tu control

Panel de administración

Aparte. Enciendes las fases de cada cliente con un clic, vía su API. No opera conversaciones.

/panel

Staging se copia → Cliente · El Panel controla las fases del Cliente

Las 4 fases (el backend trae todas; enciendes las que ya entregaste): 1 buscar por medida + cotización · 2 fitment + fotos · 3 comparativas visuales + Cotizador + Métricas · 4 Oportunidades (seguimientos y recuperación).
